Starting a charcter

When starting a character, you can start from one of two points. You can start with there backstory or you can start with their class.

Starting with the backstory is good if you already have a personality in mind for the character, and you want to make sure that the class fits the personality. Starting with the class is good if you are not compleately sure of what you want to do.

No matter where you start, the goal is to build a character that feels fun to play. If you begin with the backstory, you can shape their class around who they are as a person. If you start with the class, you can build a backstory that explains how they learned their skills. Both ways work, and many players mix the two as they play dnd
